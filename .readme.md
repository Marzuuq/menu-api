## About the Restaurant
SpiceRoute is a cozy modern Indian-fusion bistro offering flavorful mains, shareable appetizers, and indulgent desserts. We blend classic techniques with creative spice profiles.

## Project Description
This project is a simple Node.js + Express API that serves restaurant menu data and a static frontend to view it. It demonstrates API endpoints, data filtering, and a small client that fetches the API.

**Technologies used:** Node.js, Express.js, HTML, CSS, JavaScript (fetch API)
