---

## 2. Project Description
This project is a Restaurant Menu Display API built using **Node.js** and **Express.js**.  
It provides three API endpoints to retrieve menu items, vegetarian dishes, and category summaries.  
A simple frontend built with **HTML, CSS, and JavaScript (Fetch API)** allows users to browse the menu interactively.

**Technologies Used:**
- Node.js  
- Express.js  
- HTML5  
- CSS3  
- JavaScript (Fetch API)

## 5. API Documentation

### **A. GET /menu**
**Method:** GET  
**Description:** Returns all menu items available in the restaurant.

**Sample Response:**
```json
[
  {
    "id": 1,
    "name": "Margherita Pizza",
    "category": "Main Course",
    "price": 350,
    "isVegetarian": true,
    "description": "Classic pizza with tomato and mozzarella"
  }
]

B. GET /menu/vegetarian

Method: GET
Description: Returns only vegetarian menu items.

Sample Response:

[
  {
    "id": 3,
    "name": "Veg Spring Rolls",
    "category": "Appetizer",
    "price": 180,
    "isVegetarian": true,
    "description": "Crispy rolls stuffed with fresh vegetables"
  }
]

C. GET /menu/categories

Method: GET
Description: Returns unique menu categories with the number of items in each category.

Sample Response:

{
  "categories": [
    { "name": "Appetizer", "itemCount": 2 },
    { "name": "Main Course", "itemCount": 3 },
    { "name": "Dessert", "itemCount": 1 }
  ]
}

6. Installation & Setup Instructions
Step 1: Clone the Repository
git clone https://github.com/Marzuuq/menu-api.git

Step 2: Navigate to the Project Folder
cd restaurant-menu-api

Step 3: Install Dependencies
npm install

Step 4: Start the Server
npm start

Step 5: Access API Endpoints

Full menu:
http://localhost:3000/menu

Vegetarian items:
http://localhost:3000/menu/vegetarian

Categories:
http://localhost:3000/menu/categories

Step 6: Access Frontend

Open your browser and visit:

http://localhost:3000/
